## Details

**4. Set-dictionary**
* set1 & set2 means 聯集
* set1 | set2 means 交集
* set1 - set2 means 差擊
* set1 ^ set2 means 反交集
* `var in set` `var not in set` will return `True` `False`
* 從列表的資料中產生字典`dic={x:x*2 for x in [3,4,5]}` 

**5. Condition**
* 單一判斷：if ...
* 雙向判斷：if ... else
* 多條件判斷：if ... elif ... else ...

**6. Loop basic**
* for loop
* while loop

**7. Loop control**
* break continue else
* else區塊像是try-catch的finally區塊，迴圈結束前，必會執行else區塊
* 但break之後不會執行else區塊

**9. Funtion args**
* 預設參數值
* 使用參數名稱對應
* 無限(不定)參數資料(tuple)
